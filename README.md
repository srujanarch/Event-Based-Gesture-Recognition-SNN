# Event-Based-Gesture-Recognition-SNN

Completed an Event-Based Gesture Recognition Project using Spiking Neural Networks (SNNs)!


Over the past ten days, I worked on building an end-to-end gesture recognition system using the IBM DVS128 Gesture Dataset, exploring  neuromorphic computing and event-based vision.


What I built:

*Developed a custom preprocessing pipeline for AEDAT 3.1 event data.

*Converted asynchronous event streams into 30-bin voxel grid representations.

*Built and trained a Spiking Neural Network (SNN) using PyTorch and snnTorch.

*Optimized the preprocessing workflow by saving precomputed .pt tensors for faster training.

*Evaluated the model using accuracy metrics and a confusion matrix.


Results

Training Accuracy: 75.09%

Test Accuracy: 67.36%


Key Learnings

*This project was about much more than training a neural network. Along the way, I gained hands-on experience with:

*Event-based vision and neuromorphic datasets

*Custom data preprocessing and voxel-grid generation

*Debugging complex data pipelines

*Working with Google Colab resource limitations

*Organizing large ML workflows into modular preprocessing, training, and evaluation stages
